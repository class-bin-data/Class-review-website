计算过程

1. 学年划分与周数

• 七年级：2023年9月～2024年7月（40周）

• 八年级：2024年9月～2025年7月（提前开学1周，共41周）

• 九年级：2025年9月～2026年6月（提前开学1周，第二学期较短，共39周）

2. 每周在校天数

• 七、八年级：每周休息2天 → 每周在校5天

• 九年级：每周休息1天 → 每周在校6天

3. 总在校天数计算

• 七年级：40周 × 5天 = 200天

• 八年级：41周 × 5天 = 205天

• 九年级：39周 × 6天 = 234天

总天数 = 200 + 205 + 234 = 639天

4. 每日在校时间

• 七、八年级：6:40～20:30 → 13小时50分钟（13.833小时）

• 九年级：6:30～21:00 → 14小时30分钟（14.5小时）

5. 总在校时间计算

• 七年级：200天 × 13.833小时 = 2,766.6小时

• 八年级：205天 × 13.833小时 = 2,835.8小时

• 九年级：234天 × 14.5小时 = 3,393小时

总小时数 = 2,766.6 + 2,835.8 + 3,393 = 8,995.4小时

注： 计算基于标准校历，实际天数可能因节假日调整略有变化。九年级第二学期按18周计算（至中考结束）。
